QuickDraft lets you import `.txt` files from your device directly into the editor. This turns any local text file into an online, shareable draft instantly — no copy-pasting required.

## Import a .txt file

<Steps>
  <Step title="Open the editor">
    Go to [https://www.quickdraft.in](https://www.quickdraft.in) to open the editor.
  </Step>
  <Step title="Click the import icon">
    Click the **📁** (folder) icon in the toolbar.
  </Step>
  <Step title="Select your file">
    A file picker dialog opens. Navigate to your `.txt` file on your device and select it.
  </Step>
  <Step title="Edit your draft">
    The file contents load into the editor immediately. Your draft is saved to local storage automatically — you can start editing right away.
  </Step>
</Steps>

## After importing

Once your file is loaded into the editor, you can work with it just like any other draft:

- **Edit freely** — modify the content as you would any draft. Auto-save keeps every change.
- **Set a custom URL slug** — give the draft a memorable slug (for example, `meeting-notes`) to make it easy to find and share.
- **Share with others** — click **Copy Edit Link** to share a link that lets others edit the draft, or **Copy Read-Only Link** to share a view-only version.

## File format support

QuickDraft currently supports **plain text (`.txt`) files** only. The following formats are not supported for import:

| Format | Supported |
|---|---|
| Plain text (`.txt`) | ✅ Yes |
| Word document (`.docx`) | ❌ No |
| Rich Text Format (`.rtf`) | ❌ No |
| PDF (`.pdf`) | ❌ No |

<Note>
  Markdown files (`.md`) are not supported by the file importer, but you can paste Markdown content directly into the editor and preview it using the **Markdown Preview** mode. See the Markdown guide for details.
</Note>

<Tip>
  Before clearing your browser storage, export important drafts as raw text: switch to the **Raw Text** view and click **Copy All** to copy the full content to your clipboard, then paste it into a local file. This gives you a backup you can re-import into QuickDraft at any time.
</Tip>
