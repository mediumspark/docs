QuickDraft's toolbar gives you everything you need to structure and style your writing without leaving the browser. This guide covers every formatting option available.

## Text styles

Use the style selector in the toolbar to apply one of four paragraph-level formats to any block of text.

- **Normal** — the default paragraph style. Use it for body copy and general prose.
- **H1** — the top-level heading. Use it for the title or main section of your document.
- **H2** — a second-level heading. Use it to divide your document into major sections.
- **H3** — a third-level heading. Use it for subsections within an H2 section.
- **Quote** — blockquote styling. Use it to highlight citations, pull quotes, or important callouts.

To apply a style, place your cursor anywhere in a paragraph (or select multiple paragraphs), then choose the style from the dropdown.

## Inline formatting

Inline formatting lets you emphasise individual words or phrases within a paragraph.

| Style | Toolbar button | Keyboard shortcut |
|---|---|---|
| **Bold** | **B** | Ctrl/Cmd+B |
| *Italic* | *I* | Ctrl/Cmd+I |
| Underline | U | Ctrl/Cmd+U |
| ~~Strikethrough~~ | S | — |

Select the text you want to format, then click the toolbar button or press the keyboard shortcut. Click or press again to remove the formatting.

## Links

To insert a hyperlink:

1. Select the text you want to turn into a link.
2. Click the link icon (🔗) in the toolbar.
3. Enter the destination URL in the prompt that appears.
4. Press **Enter** to confirm.

To remove a link, click anywhere inside the linked text and click the link icon again.

## Lists

QuickDraft supports two list types:

- **Ordered list** (1.) — use for numbered sequences, step-by-step instructions, or ranked items. Click the ordered list icon in the toolbar to start one.
- **Bullet list** (•) — use for unordered collections of items. Click the bullet list icon in the toolbar to start one.

You can also trigger lists with Markdown syntax: start a line with `1.` for an ordered list, or `-` for a bullet list, and QuickDraft will convert it automatically.

To nest a list item, press **Tab** to indent and **Shift+Tab** to outdent.

## Keyboard shortcuts

| Action | Windows / Linux | macOS |
|---|---|---|
| Bold | Ctrl+B | Cmd+B |
| Italic | Ctrl+I | Cmd+I |
| Underline | Ctrl+U | Cmd+U |

<Tip>
  You can also write raw Markdown syntax directly in the editor — for example, `**bold**` or `## Heading 2` — and use the **+ Markdown** preview to render it as formatted output. See the [Markdown guide](/guides/markdown) for details.
</Tip>
