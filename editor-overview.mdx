The QuickDraft editor is a full-featured, browser-based writing interface designed to stay out of your way. Every control you need — from text formatting to sharing — is a single click away. This page walks through every element of the toolbar and explains exactly what each control does.

## Toolbar controls

The toolbar sits at the top of the editor and is divided into three functional rows.

### Formatting options

Apply text styles and structure using the following controls:

| Control | What it does |
|---|---|
| **Normal** | Resets the current line to unstyled body text. |
| **H1** | Formats the current line as a top-level heading. |
| **H2** | Formats the current line as a second-level heading. |
| **H3** | Formats the current line as a third-level heading. |
| **Quote** | Formats the selected text or line as a block quote. |
| **B (Bold)** | Bolds the selected text. |
| **I (Italic)** | Italicizes the selected text. |
| **U (Underline)** | Underlines the selected text. |
| **S (Strikethrough)** | Draws a line through the selected text. |

### Insert controls

| Control | What it does |
|---|---|
| **🔗 Link** | Wraps the selected text in a clickable hyperlink. You'll be prompted to enter the URL. |
| **1. Ordered list** | Starts or converts the current block into a numbered list. |
| **• Bullet list** | Starts or converts the current block into an unordered bullet list. |

### Actions row

| Control | What it does |
|---|---|
| **♡ Favorite** | Marks the draft as a favorite for quick access. |
| **🔒 Password lock** | Opens the password-protection dialog. See the [Password Protection guide](/guides/password-protection). |
| **📁 Import file** | Imports the content of a `.txt` file directly into the editor. |
| **+ New draft** | Opens a fresh, blank draft in the editor. |
| **☁ Save to account** | Syncs the current draft to your cloud account. Requires a free account. |
| **🖨 Print** | Opens your browser's print dialog for the current draft. |
| **🌙 Toggle theme** | Switches between light and dark mode. |

### Voice dictation

| Control | What it does |
|---|---|
| **🎤 Microphone** | Activates voice-to-text input. Speak to dictate content directly into the editor. See the [Voice-to-Text guide](/guides/voice-to-text) for supported browsers and tips. |

## Word and character counter

The counter bar at the bottom of the editor updates live as you write. It displays:

- **Word count** — the total number of words in your draft
- **Character count** — the total number of characters including spaces
- **Creation date** — when the draft was first created
- **Last modified date** — when the draft was most recently changed

Use the word counter to track progress against essay limits, article targets, or any length requirement.

## Share and export controls

Below the editor, a row of action buttons gives you full control over how your draft is shared and exported.

| Control | What it does |
|---|---|
| **Custom URL** | Set a custom slug for your draft's URL. See the [Custom URLs guide](/guides/custom-urls). |
| **Save Data** | Manually forces a save of the current draft to your browser's local storage. |
| **Copy Edit Link** | Copies a shareable URL that grants full editing access to anyone who opens it. |
| **Copy Read-Only Link** | Copies a shareable URL that allows viewing but not editing. |
| **Copy Share URL** | Copies the canonical URL for the current draft. |
| **Raw Text** | Displays the draft's content as plain, unformatted text. Useful for copying into other tools. |
| **Markdown Preview** | Renders any Markdown syntax in your draft as formatted output. |
| **Code View** | Displays the draft's underlying HTML source code. |

### Social sharing

Use the social sharing buttons to post a link to your draft directly from the editor. QuickDraft supports sharing to:

- **Twitter** — post a tweet with your draft link
- **Facebook** — share to your Facebook feed
- **WhatsApp** — send the link via WhatsApp chat
- **Reddit** — submit the link to a subreddit
- **Telegram** — forward the link through Telegram
- **Email** — open your default mail client with the link pre-filled

All social sharing buttons share the read-only URL of your draft, so recipients see the content without gaining editing access.

## Light and dark theme

Click the moon icon (🌙) in the toolbar to toggle between light and dark mode. QuickDraft saves your theme preference in your browser, so it persists across sessions on the same device.

## Keyboard shortcuts

Speed up your writing without reaching for the toolbar. The following shortcuts work in the editor on all platforms (use **Cmd** instead of **Ctrl** on Mac):

| Shortcut | Action |
|---|---|
| **Ctrl+B** | Bold selected text |
| **Ctrl+I** | Italicize selected text |
| **Ctrl+U** | Underline selected text |
| **Ctrl+Z** | Undo last change |
| **Ctrl+Y** | Redo last undone change |

<Tip>
  Keyboard shortcuts work even when the toolbar is hidden on smaller screens — they are always active while the editor has focus.
</Tip>
