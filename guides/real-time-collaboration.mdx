QuickDraft supports real-time collaboration — multiple people can edit the same draft at the same time. There's no account required and no setup needed. Share an edit link and your collaborators are ready to write.

## How it works

QuickDraft syncs edits across all open sessions sharing the same draft URL. As soon as one person types, everyone else viewing or editing the same draft sees the change appear instantly.

- **Instant sync** — all collaborators see changes in real time as they're typed
- **No installs required** — just a browser; no special tools, apps, or plugins needed
- **No account needed** — anyone with the link can jump straight in

## Start collaborating

<Steps>
  <Step title="Open your draft">
    Go to [quickdraft.in](https://www.quickdraft.in) to start a new draft, or navigate directly to your existing draft's URL.
  </Step>
  <Step title="Copy the edit link">
    Below the editor, click **Copy Edit Link**.
  </Step>
  <Step title="Share the link">
    Send the link to your collaborators via chat, email, or any messaging app.
  </Step>
  <Step title="Everyone starts editing">
    Collaborators open the link in their browser and start editing — changes appear for everyone in real time.
  </Step>
</Steps>

## Managing access

Anyone with the edit link can make changes to your draft. To control who can do what:

- **Restrict editing** — switch to a **Read-Only Link** so viewers can read without modifying content. See the [Sharing Links](/guides/sharing-links) guide.
- **Restrict all access** — add a **password** to your draft so only people with the password can open it at all. See the [Password Protection](/guides/password-protection) guide.

<Note>
  QuickDraft collaboration is anonymous — you won't see collaborator names or cursors. All edits are merged in real time without any identity information attached.
</Note>
