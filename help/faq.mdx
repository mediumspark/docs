This page answers the most common questions about using QuickDraft. If you don't find your answer here, see the [Troubleshooting](/help/troubleshooting) guide or contact support.

<AccordionGroup>
  <Accordion title="Do I need an account to use QuickDraft?">
    No. You can write, share, and collaborate on drafts without creating an account. An account is only needed if you want to sync your drafts across multiple devices via the cloud.
  </Accordion>

  <Accordion title="Which browsers does QuickDraft work in?">
    QuickDraft works in all modern browsers: Chrome, Edge, Safari, and Firefox. Voice-to-text dictation requires Chrome, Edge, or Safari — the Web Speech API is not supported in Firefox.
  </Accordion>

  <Accordion title="What happens to my draft if I close the tab?">
    QuickDraft auto-saves every keystroke to your browser's local storage. When you return to the same URL, your draft is restored exactly as you left it. If you clear your browser data, the draft may be lost — save important drafts to your account using **☁ Save to My Account** to avoid losing them.
  </Accordion>

  <Accordion title="Can I collaborate with multiple people at once?">
    Yes. Share the edit link with any number of collaborators. All edits are synced in real time. There's no limit on the number of simultaneous collaborators.
  </Accordion>

  <Accordion title="How do I prevent others from editing my draft?">
    Use the **Copy Read-Only Link** button to share a view-only version of your draft. You can also add password protection so only people with the correct password can view or edit the draft.
  </Accordion>

  <Accordion title="Does QuickDraft support Markdown?">
    Yes. Write Markdown syntax directly in the editor, then click **+ Markdown** below the editor to open a panel that renders the preview output.
  </Accordion>

  <Accordion title="Can I use QuickDraft offline?">
    You can continue editing a draft you've already opened while offline — changes will auto-save to local storage. However, real-time collaboration and cloud sync both require an active internet connection.
  </Accordion>

  <Accordion title="Can I import documents from Word or Google Docs?">
    Not directly. QuickDraft supports importing `.txt` files. For Word or Google Docs content, export your document as plain text first, then import the `.txt` file into QuickDraft.
  </Accordion>

  <Accordion title="Is QuickDraft free?">
    Yes. The core editor, sharing, real-time collaboration, and local storage are all free. An optional free account adds cloud sync so your drafts are available across devices.
  </Accordion>

  <Accordion title="How do I delete a draft?">
    - **Local drafts:** Open the **Saved Drafts** panel and click **Clear All Local** to remove all browser-stored drafts.
    - **Account drafts:** Open the **My Account Drafts** tab and delete individual drafts from the list.
  </Accordion>
</AccordionGroup>
