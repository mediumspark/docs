---
title: "Secure Your Draft: Add Password Protection in QuickDraft"
sidebarTitle: "Password Protection"
description: "Lock any QuickDraft draft behind a password. Only people with the password can view or edit the draft — ideal for confidential notes."
---

QuickDraft lets you lock any draft behind a password. Once protected, anyone visiting the draft URL — whether with an edit link or a read-only link — must enter the correct password before they can access the content.

## Add a password

<Steps>
  <Step title="Open your draft">
    Open your draft in the editor at [quickdraft.in](https://www.quickdraft.in).
  </Step>
  <Step title="Open the password dialog">
    Click the 🔒 lock icon in the toolbar (or the **🔒 Password Protect** option).
  </Step>
  <Step title="Enter a password">
    Type your chosen password in the dialog box.
  </Step>
  <Step title="Set the password">
    Click **Set Password**. The draft is now locked — anyone who visits the URL will see a password prompt before the content loads.
  </Step>
</Steps>

## Sharing a password-protected draft

Share the draft URL as you normally would — via **Copy Edit Link** or **Copy Read-Only Link**. The password protection applies to both link types.

- When collaborators open the link, they'll see a password prompt before the draft loads.
- You'll need to share the password with them separately — for example, via a secure message or a private channel.

<Info>
  Password protection is applied at the draft level, not the link level. Both edit and read-only links will require the password until you remove it.
</Info>

## Remove a password

<Steps>
  <Step title="Open the password dialog">
    Click the 🔒 lock icon in the toolbar.
  </Step>
  <Step title="Clear the password field">
    Leave the password field blank.
  </Step>
  <Step title="Confirm removal">
    Click **Set Password**. The password is removed and the draft becomes publicly accessible to anyone with the link.
  </Step>
</Steps>

<Warning>
  QuickDraft password protection restricts who can view the draft, but the draft URL is still guessable if you've set a short or predictable custom slug. For maximum privacy, use a random (non-custom) slug **and** a strong password.
</Warning>

<Tip>
  Choose a strong password that is at least 12 characters long and not reused from other services. Combining uppercase letters, numbers, and symbols makes your draft significantly harder to access without authorisation.
</Tip>
